# ybmp
your basic music player is a music player written in html, css &amp; js. it uses the indexeddb library to, in a fully offline way, store playlists made by users, where they use an image as a cover, and audio/video files as the content to play. it features a hell lotta stuff &amp; more to come, check it out! no servers, no accounts, just you &amp; whats yours.
